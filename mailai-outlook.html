How to set up your Azure app (5 minutes, free)
1
Go to portal.azure.com → search App registrations → click New registration
2
Name it anything (e.g. MailAI). Under Supported account types pick Accounts in any organizational directory and personal Microsoft accounts
3
Under Redirect URI choose Mobile and desktop applications and tick or enter: https://login.microsoftonline.com/common/oauth2/nativeclient — this lets popup login work from any file on your computer with no extra setup.
4
Click Register. Copy the Application (client) ID from the Overview page and paste it above.
5
Go to API permissions → Add permission → Microsoft Graph → Delegated → add: Mail.Read, Mail.Send, User.Read → click Grant admin consent
